# 桂工助手

<img src="screenshot/logo.png" align="right" title="GlutAssistant" width="120" height="120">

适用于桂林理工大学教务系列的 Android APP，包含了各种常用的教务查询。

## 功能

* 课程表(支持课程管理)
* 成绩查询(课程成绩，体侧成绩)
* 考试地点查询
* 支持教务登录，统一身份认证登录，南宁校区
